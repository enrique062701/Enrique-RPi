This README.TXT file contains information which helps you to start software.


Following devices may by controlled remotely by software:
=========================================================
Laser PL225X + OPO PG4XX, PG5XX, Ekspla Ltd, without software modification 
Other lasers models may be controled by replacing names of the modules and registers in the source code.

Software package contents:
=========================
1. Sources of Demo application writen in Delphi 2009

Minimum Hardware and Software Requirements
==========================================
  *   Pentium processor
  *   Microsoft Windows XP/2000/Wista/Windows 7
  *   CD-ROM drive (for installation only)
  *   USB port.


Installing
=========

1.Install of test and user utility 'CAN browser'. This step installs USB drivers.
2.Copy folder contens to your hard disk.
3.Copy product specific files: REMOTECONTROL.CSV and REMOTECONTROL.INI to the same folder. Files you will find in control panel application directory: /CAN network/Control panel appications/'Your product name'

Notes
=========

1. Visibility of the amplification ON/OFF control is turned off. In case of laser model PL2251 (with amplifier) you may turn the control on.
