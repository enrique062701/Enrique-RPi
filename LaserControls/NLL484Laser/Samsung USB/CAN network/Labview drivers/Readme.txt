This README.TXT file contains information which helps you to start software.


Following devices may by controlled remotely by software:
=========================================================
All Ekspla products may be controled by replacing names of the modules and registers in the source code.

Software package contents:
=========================
1.LabVIEW drivers 
2. REMOTECONTROL.dll file
3. USBCAN.dll file
4. CANRS232.dll file
5. REMOTECONTROL.CSV
6. REMOTECONTROL.INI
7. ERRORS.TXT

Minimum Hardware and Software Requirements
==========================================
  *   Pentium processor
  *   Microsoft Windows XP/2000/Wista/Windows 7
  *   CD-ROM drive (for installation only)
  *   USB port.


Installing
=========
Following steps must be made:

1.Installing of test and user utility 'CAN browser'. This step installs USB drivers.
2.In case you have LabView 7.1.1 or later version installed.


Starting for the first time
===========================

   
  * Copy Labview drivers folder to your computer.
  * Disconnect CAN browser.
  * To run LabView files from LabView software, select ..\\Labview drivers\LWxxx\Remotecontrol.llb library. 
  * Run Getting Started.vi
